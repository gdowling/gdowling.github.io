Installation

The easiest way to install python is to use Anaconda. Download the 64-bit version of Python 3.7.

Anaconda will come with four main components:

  1. Python 3.7: this allows you to run Python
  2. Spyder: An Integrated Development Environment (IDE) similar to R-Studio. This is where you will work on most of your projects.
  3. Jupyter Notebook: Jupyter Notebook is a web-based interactive computational environment where you can write Python code. Popular because you can include easy to read text around you code to explain your process. Similar to R-Markdown.
  4. Anaconda Prompt: This is were you will go to download various Python packages. Similar to how you download packages in R (Will cover more on this later).
