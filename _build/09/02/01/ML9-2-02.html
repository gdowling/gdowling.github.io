Multiple Regression

In most cases you will want to use multiple independent variables to predict a dependent variable. This is not a problem at all for you will just add more X variables to the equation.

Y = m1X1 + m2X2 + m3X3 + b

Something to note when adding more variables to your regression is multicollinearity. This exists whenever an independent variable is highly correlated with one or more of the other independent variables in a multiple regression equation. This is a problem because it undermines the statistical significance of an independent variable. A way to prevent this is by checking the Variance Inflation Factor for each variable.
